* **UI Server**: React SPA served by Nginx
* **Gateway**: REST API for admin operations
* **WebSocket**: Real-time agent communication

== Authentication

=== JWT Configuration

.Authentication uses JWT tokens with configurable secrets.

[source,bash]
----
export JWT_SECRET=your-32-character-secret-here
export JWT_EXPIRY_HOURS=24
----

.Sessions are stored in Redis:

[options="header", cols="2,2,3"]
|===

== Real-Time Features

=== WebSocket Connection

.Real-time updates via WebSocket:

[source,javascript]
----
const ws = new WebSocket('wss://your-domain.com/ws');

ws.on('agent_status', (data) => {
  console.log('Agent status update:', data);
});

ws.on('request_complete', (data) => {
  console.log('Request completed:', data);
});
----

=== Event Types

.Subscribe to events:

[cols="2,3"]
|===
| Event | Payload
| `agent.created`
| Agent registration details

| `agent.status`
| Status changes (idle, running, error)

| `request.started`
| Request metadata and ID

| `request.completed`
| Response with tokens and cost

| `request.failed`
| Error details
|===

=== Live Updates

.Dashboard automatically updates:
* Agent list refreshes every 5 seconds
* Metrics graph updates every second
* Log stream follows agent activity

== Production Configuration

=== Reverse Proxy

.Nginx configuration for production:

[source,nginx]
----
server {
    listen 443 ssl;
    server_name your-domain.com;

    ssl_certificate /etc/ssl/certs/server.crt;
    ssl_certificate_key /etc/ssl/private/server.key;

    location / {
        proxy_pass http://ui:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    location /ws {
        proxy_pass http://ws:8081;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
    }
}
----

=== Static File Serving

.Build and serve static assets:

[source,bash]
----
# Build static files
docker compose run ui npm run build

# Serve with Nginx
location /static/ {
    alias /app/static/;
    expires 30d;
}
----

== Troubleshooting

=== Common Issues

.UI won't load:
* Check `docker compose logs ui`
* Verify JWT secret is set
* Check browser console for errors

.WebSocket disconnected:
* Verify `ws` service is running
* Check firewall settings for port 8081
* Verify TLS configuration for WSS

=== Debug Mode

.Enable debug logging:

[source,bash]
----
export GASTOWN_LOG_LEVEL=debug
export UI_DEBUG=true
----
